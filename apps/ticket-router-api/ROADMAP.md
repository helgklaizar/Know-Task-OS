# ROADMAP

## General Objective
Go from Python basics to creating a full-fledged Production-ready ML system.

### Phase 1: Data Foundations (Weeks 1-2)
- [ ] Set up the environment and explore Python infrastructure (pip, venv/conda, jupyter).
- [ ] Advanced Python (typing, OOP, generators).
- [ ] Data collection: web scraping (BeautifulSoup) and API integration.
- [ ] Preprocessing and analysis (EDA): NumPy, Pandas. Saving to SQL/PostgreSQL.
- [ ] Visualization: Matplotlib, Seaborn, interactive Plotly.

### Phase 2: Classic Machine Learning (Weeks 3-4)
- [ ] Classic algorithms: Linear/Logistic Regression, KNN, SVM (scikit-learn).
- [ ] Decision Trees and Ensembles: Random Forest, XGBoost, LightGBM, CatBoost.
- [ ] Cross-validation, hyperparameter tuning (Optuna).
- [ ] Quality metrics (ROC-AUC, F1, RMSE).

### Phase 3: Deep Learning (Weeks 5-6)
- [ ] Introduction to PyTorch: tensors, gradients, neural network architecture.
- [ ] CV (Computer Vision): Convolutional Neural Networks (CNN), ResNet, Transfer Learning.
- [ ] NLP: Recurrent Neural Networks (LSTM), Attention mechanism.
- [ ] Hugging Face Transformers (BERT and equivalents).

### Phase 4: MLOps & Production (Weeks 7-8)
- [ ] Final ML system (project selected from implementation_plan).
- [ ] Wrapping the model in a REST API (FastAPI).
- [ ] Docker containerization of the service.
- [ ] UI creation (Streamlit).
